Integrated Non-Invasive Assessment of Kidney Health

My Pre Final Year Project

This innovative system combines the power of urinalysis, imaging studies, and bioimpedance spectroscopy. By leveraging machine learning algorithms, specifically the Support Vector Machine (SVM) algorithm, the system analyzes data from these modalities to provide accurate assessments of kidney health. It includes functionalities for automated report generation and a user-friendly web dashboard tailored for healthcare professionals, facilitating informed clinical decisions.

Technologies & Tools Used:
Machine Learning- Support Vector Machine (SVM) Algorithm, User Interface Design

Fisherman Border Cross Alert System Using LoRa

Mini Project

This innovative system utilizes LoRa (Long Range) technology for wireless communication between nodes installed on fishing boats and border stations. The nodes communicate using the SPI (Serial Peripheral Interface) protocol to transmit distance data obtained from GPS sensors. By leveraging LoRa's low-power, long-range capabilities, the system ensures reliable and efficient communication even in remote maritime environments. The implementation includes robust data handling and processing on microcontroller platforms such as the Atmega 328p, programmed using the Arduino IDE.

Technologies & Tools Used:
LoRa02 module, Atmega 328p microcontroller, Arduino IDE, GPS sensor
